# weather-dashboard




## Description
This is a weather app that displays the current weather for whatever city the user enters as well as the forecast for the next five days. 

## Build With
* JavaScript
* HTML
* CSS
* Public APIs:
  - https://api.nasa.gov/
  - https://aztro.readthedocs.io/en/latest/
